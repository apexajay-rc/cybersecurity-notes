# Enumeration Checklist

# Initial Recon

- Identify target IP/domain
- Check host availability
- Identify open ports
- Fingerprint services
- Identify technologies/frameworks

---

# Web Enumeration

- Review HTTP headers
- Inspect page source
- Identify login panels
- Enumerate directories
- Check robots.txt
- Check technologies used
- Identify forms/input fields

---

# SMB Enumeration

- Enumerate shares
- Check anonymous access
- Enumerate users/groups

---

# Linux Enumeration

- Kernel version
- Running services
- SUID binaries
- Cron jobs
- Writable files/directories
- Environment variables

---

# Windows Enumeration

- Users/groups
- Installed software
- Services
- Scheduled tasks
- Token privileges

---

# Notes

Always enumerate deeper before exploiting.
