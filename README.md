# Cybersecurity Notes

Structured knowledge base documenting concepts, methodologies, tooling, practical labs, and security research across multiple areas of cybersecurity and penetration testing.

This repository is maintained as a continuously evolving technical reference for:
- system internals
- networking
- web application security
- enumeration methodologies
- privilege escalation
- attack surface analysis
- offensive security tooling
- lab documentation and experimentation

The primary focus of this repository is practical understanding, reproducibility, and methodology-driven learning.

---

# Repository Structure

## Linux
Linux fundamentals, permissions, processes, services, filesystem behavior, shell usage, and privilege escalation concepts.

## Networking
TCP/IP, DNS, routing, ports, protocols, packet analysis, and network enumeration techniques.

## Web Security
HTTP fundamentals, authentication mechanisms, session handling, common web vulnerabilities, and exploitation workflows.

## Enumeration
Reconnaissance methodologies, service fingerprinting, attack surface mapping, and information gathering techniques.

## Methodology
Structured workflows, checklists, operational notes, and repeatable assessment approaches.
